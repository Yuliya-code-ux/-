# Семейный бюджет

Небольшой учебный проект с приветственным лендингом и калькулятором семейного бюджета.

## Что есть в проекте

- приветственный блок с описанием и быстрыми действиями;
- калькулятор доходов и расходов семьи;
- расчет остатка бюджета и доли расходов от дохода;
- подсказка по рекомендуемому резерву (20% дохода);
- история последних расчетов (сохраняется в `localStorage`).

## Структура

- `index.html` — точка входа (редирект на основную страницу);
- `калькулятор семейного бюджета.html` — основной лендинг + калькулятор;
- `assets/styles.css` — стили интерфейса;
- `assets/app.js` — логика калькулятора и истории.

## Запуск

1. Откройте `index.html` в браузере.
2. Или откройте сразу `калькулятор семейного бюджета.html`.
